
## The code prints the sequence of how to light three leds on ESP32 IDF
### Hardware Required

* A development board with ESP32/ESP32-S2/

| Board                | LED type             | Pin                  |
| -------------------- | -------------------- | -------------------- |
| ESP32                | LED RED              | GPI012               |
| ESP32                | LED GREEN            | GPI014               |
| ESP32                | LED WHITE            | GPIO26               |
